# Socket-Based People Management System

This project was created for my Principles of Programming Languages class (COMP348 - Winter 2025).

It is built using Python and demonstrates concepts such as
- data validation
- client-server communication

## Features
- read and add people from a data.txt file
- add, delete or modify people's information in the database
- display list of people alphabetically with their details

## How to Run
1. download or clone the repository
2. open two terminal windows
3. in the first terminal, start the server by running: python3 server.py
4. in the second terminal, start the client by running: python3 client.py
